* Installation
#+BEGIN_SRC shell
curl https://dotfiles.codearsonist.com/install.sh | sh
#+END_SRC
or with nix:
#+BEGIN_SRC shell
nix build -f https://github.com/peel/dotfiles/archive/master.tar.gz
./result/bin/dotfiles
#+END_SRC
* Usage
#+BEGIN_SRC shell
dotfiles [command]
#+END_SRC
- ~default~ (~install~)
- ~link~ - link/relink dotfiles
- ~uninstall~ - remove configured links
- ~unlink~ - unlink dotfiles
