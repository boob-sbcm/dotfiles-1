#+BEGIN_SRC shell

./result/bin/dotfiles
#+END_SRC
* Usage
#+BEGIN_SRC shell
dotfiles [command]
#+END_SRC
- ~default~ (~install~)
- ~link~ - link/relink dotfiles
- ~uninstall~ - remove configured links
- ~unlink~ - unlink dotfiles
